Resumen Ejecutivo
Desarrollo de una aplicación web para GreenSeller que automatiza el análisis de datos climáticos en invernaderos, generando predicciones precisas del día de corte de rosas mediante algoritmos personalizados para cada variedad y cliente.
Introducción
Propuesta para crear una aplicación web que automatice el análisis de datos agrícolas de GreenSeller, optimizando la gestión de cultivos de rosas mediante tecnología sostenible e inteligencia digital aplicada.
Planteamiento del Problema
GreenSeller analiza manualmente datos climáticos de invernaderos, lo que retrasa la predicción del día óptimo de corte y limita su eficiencia. Se necesita una herramienta digital que integre datos y genere resultados precisos de forma automática.
Solución Propuesta
Desarrollar una app web en Laravel que conecte la API de WeatherLink, procese datos en tiempo real y genere gráficos y predicciones personalizadas por variedad de rosa, optimizando la productividad y la sostenibilidad del cultivo.
